Resolve Card BIN

```text [GET]
{{baseUrl}}/api/v1/general/bin?card_bin
```

| GET Parameter | Type | Description |
| --- | --- | --- |
| card\_bin _required_ | int | First 6 digits of your card |

```json Response
{
  "entity": {
    "bin": "506118",
    "brand": "Verve",
    "sub_brand": "",
    "country_code": "NG",
    "country_name": "Nigeria",
    "card_type": "DEBIT",
    "bank": "Ecobank Nigeria"
  }
}
```
