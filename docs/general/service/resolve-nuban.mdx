### Request

```text [GET]
{{baseUrl}}api/v1/general/account
```

### Header

| Parameter | Type | Description |
| --- | --- | --- |
| AppId | string | you would need to create an app to get your app ID |
| Authorization | string | private key e.g prod\_sk\_1T1eSavlZxy02OT3OWlvbxK4G |

### Query parameter

| Parameter | Type | Description |  |
| --- | --- | --- | --- |
| account\_number \* | string | A valid NUBAN | required |
| bank\_code \* | integer | Bank Code | required |
