---
title: "SA National ID"
mode: "wide"
description: "This endpoint allows you to Lookup a South African National ID."
---


### Request

```text [GET]
{{baseURL}}/api/v1/za/kyc/id
```

### Header

| Parameter | Type | Description |
| --- | --- | --- |
| AppId | string | you would need to create an app to get your app ID |
| Authorization | string | public secret key e.g prod\_sk\_1T1eSavlZxy02OT3OWlvbxK4G |

### Query partner

| Parameter | Type | Description |  |
| --- | --- | --- | --- |
| id\_number \* | string | South African National ID Number | required |
| with\_photo | boolean | default is false |  |
