---



### Request

```text [GET]
{{baseUrl}}/api/v1/kyc/nuban/status
```

### Header

| Parameter | Type | Description |
| --- | --- | --- |
| AppId | string | you would need to create an app to get your app ID |
| Authorization | string | public secret key e.g prod_sk_1T1eSavlZxy02OT3OWlvbxK4G |

### Query parameters

| Parameter | Type | Description |  |
| --- | --- | --- | --- |
| account_number \* | string | Bank account number | required |
| bank_code \* | integer | Bank Code | required |
