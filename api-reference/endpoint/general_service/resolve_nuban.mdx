What is a NUBAN?

NUBAN means Nigerian Uniform Bank Account Number. It is the standardized system for numbering bank accounts in Nigeria

Resolve NUBAN is an endpoint fetches all the information associated with a NUBAN


### Response

<ResponseField name="wallet_balance" type="string">
    Wallet balance 
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://api.dojah.io/api/v1/general/account' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--header 'AppId: AppId <appId>' \

```
