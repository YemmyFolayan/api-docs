### Request

```text [GET]
{{baseURL}}/api/v1/zw/kyc/fcb
```

### Header

| Parameter | Type | Description |
| --- | --- | --- |
| AppId | string | you would need to create an app to get your app ID |
| Authorization | string | public secret key e.g prod\_sk\_1T1eSavlZxy02OT3OWlvbxK4G |

### Query parameter

| Parameter | Type | Description |  |
| --- | --- | --- | --- |
| id\_number \* | string | Zimbabwian National ID | required |
| dob \* | string | date of birth | required |
| name \* | string |  | required |
| surname \* | string |  | required |
| gender\* | string | M or F | required |
| marital\_status\* | string | M or S | required |
| mobile\_number\* | string |  | required |
