**PERFORM SINGLE LOOKUP** The Individual Single Lookup feature enables businesses to conduct a user lookup using the following steps:

1) To initiate an Individual Single Lookup, businesses should click on the "Perform Individual Single Lookup Button", triggering the appearance of a **side modal**.

![](https://guides.dojah.io/~gitbook/image?url=https%3A%2F%2F3296995301-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F2pgHlRJB7EejHn55UcwX%252Fuploads%252FJzQqtAzVeot7n0g1gtDf%252FScreenshot%25202024-06-03%2520at%252016.09.42.png%3Falt%3Dmedia%26token%3Dd71aec65-472d-414d-acc0-0df6e660472e&width=768&dpr=4&quality=100&sign=799f4e9d&sv=1)

Side modal for Individual single Lookup

2) Businesses must select a **country** and **app,** and subsequently choose the **ID type** from a dropdown menu that includes various IDs based on the selected country.

![](https://guides.dojah.io/~gitbook/image?url=https%3A%2F%2F3296995301-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F2pgHlRJB7EejHn55UcwX%252Fuploads%252FJ8aLMcW1VymchG3YgMQh%252Fimage.png%3Falt%3Dmedia%26token%3D1b46659c-34a6-475e-a927-d354acfaed79&width=768&dpr=4&quality=100&sign=b17be1fc&sv=1)

Select the "country" drop-down option

![](https://guides.dojah.io/~gitbook/image?url=https%3A%2F%2F3296995301-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F2pgHlRJB7EejHn55UcwX%252Fuploads%252FgermUebuK3cvgDnyMIlD%252Fimage.png%3Falt%3Dmedia%26token%3D67818157-11aa-4073-9339-e9bce744349b&width=768&dpr=4&quality=100&sign=674951ab&sv=1)

Select the "IDs type" drop-down option

**PERFORM BATCH LOOKUP**

For a business with data on a multitude of users seeking to verify their identities through Individual Lookup, they can achieve this via Individual Batch Lookup by following the steps outlined below:

1) To initiate Individual Batch Lookup, businesses should click on the "Perform Batch Lookup" button, triggering the appearance of the **side modal** as illustrated below.

![](https://guides.dojah.io/~gitbook/image?url=https%3A%2F%2F3296995301-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F2pgHlRJB7EejHn55UcwX%252Fuploads%252FruF8Kj7fdkHDESCgYAeG%252Fimage.png%3Falt%3Dmedia%26token%3Dc193b126-860b-4fe9-b5bb-a5aa07dc33a8&width=768&dpr=4&quality=100&sign=7467dd63&sv=1)

Side modal for performing Batch Lookup

2) Businesses are required to choose a **country** from a dropdown menu, select an **app**, and then choose an **ID type** from a dropdown menu. The available options for the ID type depend on the country selected

![](https://guides.dojah.io/~gitbook/image?url=https%3A%2F%2F3296995301-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F2pgHlRJB7EejHn55UcwX%252Fuploads%252FBO4Ii7gXoyHrdT515ANT%252Fimage.png%3Falt%3Dmedia%26token%3D04df2fff-511b-4c59-929c-bcddeb6d7120&width=768&dpr=4&quality=100&sign=e18922d8&sv=1)

Select a country with drop-down options

![](https://guides.dojah.io/~gitbook/image?url=https%3A%2F%2F3296995301-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F2pgHlRJB7EejHn55UcwX%252Fuploads%252FtoKZk7nzbTZhLNPj9QQV%252Fimage.png%3Falt%3Dmedia%26token%3Db9b7a463-acbe-40c7-9a82-4480f08bb28a&width=768&dpr=4&quality=100&sign=b863f29f&sv=1)

Select an app to use for the verification process

![](https://guides.dojah.io/~gitbook/image?url=https%3A%2F%2F3296995301-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F2pgHlRJB7EejHn55UcwX%252Fuploads%252F1XJaj6ptdIcuysrRKWER%252Fimage.png%3Falt%3Dmedia%26token%3D61c6b476-69e8-4bcc-8017-f1b616bd9200&width=768&dpr=4&quality=100&sign=e0f1bfc8&sv=1)

Select an ID type available for the country chosen

3) Businesses have the flexibility to upload user data in their preferred format or download the template by clicking on "click here to select." They can then populate the template with users data and proceed to upload it.

![](https://guides.dojah.io/~gitbook/image?url=https%3A%2F%2F3296995301-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F2pgHlRJB7EejHn55UcwX%252Fuploads%252FOwoxuamABZYF1E1M2HzD%252Fimage.png%3Falt%3Dmedia%26token%3Dd90f1227-6d60-4b6b-8635-c01ba5c65eb5&width=768&dpr=4&quality=100&sign=615b458b&sv=1)
