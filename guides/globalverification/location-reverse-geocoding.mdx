---
title: "Location Reverse GeoCoding"
mode: "wide"
description: "Get approximate address of user's location"
---


```text [GET]
{{baseUrl}}/api/v1/kyc/address/reverse_geocode
```

| QUERY Parameter | Type | Description |
| --- | --- | --- |
| longitude | string | longtitude of location |
| latitude | string | latitude of the location |

### Header

| Parameter | Type | Description |
| --- | --- | --- |
| AppId | string | you would need to create an app to get your app ID |
| Authorization | string | public secret key e.g prod\_sk\_1T1eSavlZxy02OT3OWlvbxK4G |
