---
title: "Fetch Address Verification Data"
mode: "wide"
description: "Retrieve the details of an address verification requested"
---



```text [GET]
{{baseUrl}}/api/v1/kyc/address
```

### Header

| Header | Type | Description |
| --- | --- | --- |
| AppId | string | you would need to create an app to get your app ID |
| Authorization | string | public secret key |

### Query parameter

| Key | Type | Description |
| --- | --- | --- |
| reference\_id | string | 69e10264-4b90-64fe-b4b7-c9dddafd0241 |
