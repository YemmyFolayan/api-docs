### Request

```text [GET]
{{baseUrl}}/api/v1/kyc/nuban
```

### Header

| Parameter | Type | Description |
| --- | --- | --- |
| AppId | string | you would need to create an app to get your app ID |
| Authorization | string | public secret key e.g prod\_sk\_1T1eSavlZxy02OT3OWlvbxK4G |

### Query parameters

| Parameter | Type | Description |  |
| --- | --- | --- | --- |
| account\_number \* | string | Bank account number | required |
| bank\_code \* | integer | Bank Code | required |
