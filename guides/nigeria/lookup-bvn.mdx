### [](https://api-docs.dojah.io/docs/lookup-bvn#request)

```text [GET]
{{baseUrl}}/api/v1/kyc/bvn/full</span>

```

### [](https://api-docs.dojah.io/docs/lookup-bvn#headers)

| Parameter | Type | Description |
| --- | --- | --- |
| AppId | string | you would need to create an app to get your app ID |
| Authorization | string | public secret key e.g prod\_sk\_1T1eSavlZxy02OT3OWlvbxK4G |

### [](https://api-docs.dojah.io/docs/lookup-bvn#query-parameters)

| Parameter | Type | Description |  |
| --- | --- | --- | --- |
| bvn \* | string | A valid bvn | required |

### [](https://api-docs.dojah.io/docs/lookup-bvn#response-sample)
