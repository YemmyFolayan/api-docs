## Lookup Voters ID


### Request

```text [GET]
{{baseUrl}}/api/v1/kyc/vin
```

### Header

| Parameter | Type | Description |
| --- | --- | --- |
| AppId | string | you would need to create an app to get your app ID |
| Authorization | string | public secret key e.g prod_sk_1T1eSavlZxy02OT3OWlvbxK4G |

### Query parameters

<table id="table"><tbody><tr><td>vin</td><td>string</td><td>Voter's identification number</td><td>optional</td></tr></tbody></table>
