---
title: "Fetch All Webhooks"
mode: "wide"
description: "This endpoint allows you to Fetch All your Application Webhooks"
---


###Request

```text [GET]
 {{baseUrl}}api/v1/webhook/fetch
```

### [](https://api-docs.dojah.io/docs/subscribe-to-services#header)

| Parameter | Type | Description |
| --- | --- | --- |
| AppId | string | you would need to create an app to get your app ID |
| Authorization | string | public secret key e.g prod\_sk\_1T1eSavlZxy02OT3OWlvbxK4G |
