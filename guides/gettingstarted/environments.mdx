You have access to two environments: **Sandbox** and **Live.**

You can easily build in the sandbox environment and move to live using the same API endpoints when production ready.

From the dashboard, you can get the sandbox and live keys to use the services

| Environments | Base URL |
| --- | --- |
| Sandbox | [https://sandbox.dojah.io/](https://sandbox.dojah.io/) |
| Live | [https://api.dojah.io/](https://api.dojah.io/) |
